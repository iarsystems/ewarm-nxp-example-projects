@Example description:
	Purpose:
		This example describes how to use Capture Timer function.
	Process:
		We use Timer 0 to take a snapshot of the timer value when an input signal
		on CAP0.2 (pin PF.8) transitions.
		Timer configuration:
			- Prescaler in microsecond value
			- prescaler value = 1000000us = 1s
			- Use channel, CAPn.0
			- Enable capture both on rising and falling edge
			- Generate capture interrupt
		Whenever capture interrupt occurs, TIMER interrupt service routine will be invoke
		to get captured time and display it into serial display
		Changing connect PF_8 with GND and VCC whenever want to capture time.

@Directory contents:
	\IAR:	includes IAR EWARM project and configuration files
	lpc43xx_libcfg.h: Library configuration file - include needed driver library for this example
	Timer_Capture.c: Main program

@How to run:
	Hardware configuration:
		This example was tested on:
			Hitex LPC4350 Board

	Serial display configuration: (e.g: TeraTerm, Hyperterminal, Flash Magic...)
		– 115200bps
		– 8 data bit
		– No parity
		– 1 stop bit
		– No flow control

	Running mode:
		RAM mode:   This example can be run on RAM mode with debugger.
		ROM mode:   This example can be run on EXFLASH mode with debugger.

	Step to run:
		- Step 1: Build example.
		- Step 2: Connect UART1 on this board to COM port on your computer
		- Step 3: Configure hardware and serial display as above instruction
		- Step 4: Start Debug then Run example and observe timer capture value on serial display
		Changing connect PF_8 with GND and VCC whenever want to capture time.

		(Pls see "LPC43xx Example Description" document - chapter "Examples > TIMER > Capture"
		for more details)
