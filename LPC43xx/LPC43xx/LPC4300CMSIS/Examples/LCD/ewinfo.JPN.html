Example description

  
This example describes how to config LCD on Hitex board and LogicPD 4.3.

Example description:
	Purpose:
		This example describes how to config LCD on Hitex board and LogicPD 4.3.
	Process:
		- Set up LCD panel clock
		- Init SDRAM and SRAM
		- Init IO port for LCD
		- Init LCD Controller
		- Fill Colorbar and Logo
		*** Work with default jumper ***

Directory contents:
	\IAR:	includes IAR EWARM project and configuration files
	lpc43xx_libcfg.h: Library configuration file - include needed driver library for this example
	IS42S16400D.c: Support functions for intializing I242S16400D IC
	Lcd_Demo.c: Main program

How to run:
	Hardware configuration:
		This example was tested on:
			Hitex LPC4350 board

	Running mode:
		RAM mode:   This example can be run on RAM mode with debugger.
		ROM mode:   This example can be run on EXFLASH mode with debugger.

	Step to run:
		- Step 1: Build example.
		- Step 2: Burn hex file into board (if run on ROM mode)
		- Step 3: Run example
