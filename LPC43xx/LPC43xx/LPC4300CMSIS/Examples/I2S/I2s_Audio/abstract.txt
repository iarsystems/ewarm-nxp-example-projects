@Example description:
	Purpose:
		This example describes how to use I2S to transfer audio data through I2S port
		to play a short music sound.
	Process:
		To run on ex flash, be careful on setting Core Clock which will relate to Flash Timing
		I2S setup:
			- wordwidth: 16 bits
			- stereo mode
			- master mode for TX
			- frequency = 48Khz (maximum is 96kHz)
			- enable transmit interrupt mode
			- Irq TXFIFO depth = 4
		Audio data will be copy in tx_buffer to send to TXFIFO register, after half of
		buffer have been sent already, call I2S_Callback() function to re-fill data.

@Directory contents:
	\IAR:	includes IAR EWARM project and configuration files
	audiodata.c: store audio data
	I2s_Audio.c: Main program

@How to run:
	Hardware configuration:
		This example was tested on:
			Hitex LPC4350 Board

	Running mode:
		RAM mode:   This example can be run on RAM mode with debugger.
		ROM mode:   This example can be run on EXFLASH mode with debugger.

	Step to run:
		- Step 1: Build example.
		- Step 2: Burn hex file into board (if run on ROM mode)
		- Step 4: Configure hardware and serial display as above instruction
		- Step 5: Run example and hear short music via headphone
