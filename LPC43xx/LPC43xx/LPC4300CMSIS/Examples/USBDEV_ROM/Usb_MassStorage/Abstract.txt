@Example description:
	Purpose:
		This example describes how to write a USB MassStorage
		application using USB ROM driver on LPC43xx.

@Directory contents:
	\IAR:	includes IAR EWARM project and configuration files
	usbd_msd_mem.c : contains USB MSC function and memory device definition
	msc_usbdesc.c : contains usb device descriptor.

@How to run:
	Hardware configuration:
		This example was tested on:
			Hitex LPC4350 Development Board

	Running mode:
		RAM mode:   This example can be run on RAM mode with debugger.
		ROM mode:   This example can be run on EXFLASH mode with debugger.

		(Please reference "LPC1000 Software Development Toolchain" - chapter 4 "Creating and working with
		LPC1000CMSIS project" for more information)

	Step to run:
		- Step 1: Build example.
		- Step 2: Burn hex file into board (if run on ROM mode)
		- Step 3: Configure hardware as above instruction
		- Step 4: Run example
		- Step 5: Connect the USB0 port to PC with USB cable.
		- Step 6:
			- Testing Mass storage interface
					- On PC a new unformatted drive with 32KB size should appear.
					- Format the drive and copy images to the drive.

		(Pls see "LPC43xx Example Description" document - chapter "Examples > USBDEV_ROM > USBComposite"
		for more details)
