Deep PowerDown example

Example description:
	Purpose:
		This example describes how to enter system in Deep PowerDown mode and wake-up
		by using RTC (Real-time clock) interrupt
	Process:
		When system enter in Deep PowerDown mode, it just can be waked up when an external
		reset signal is applied, or the RTC interrupt is enabled and an RTC interrupt is
		generated.
		In this case, we can use both RTC interrupt or hit RESET button to wake-up system
		RTC configure:
			- Alarm time: 5s
		So, after each 5s, RTC will generate Alarm interrupt to wake up system.

		After flash code, press reset button and wait.

Directory contents:
	\IAR:	includes IAR EWARM project and configuration files
	lpc43xx_libcfg.h: Library configuration file - include needed driver library for this example
	Pwr_DeepPowerDown.c: Main program

How to run:
	Hardware configuration:
		This example was tested on:
			Hitex LPC4350 Board

		Serial display configuration: (e.g: TeraTerm, Hyperterminal, Flash Magic...)
			– 115200bps
			– 8 data bit
			– No parity
			– 1 stop bit
			– No flow control
			- Use USB cable connect to board's UART1->USB port

	Running mode:
		ROM mode:   This example can be run on EXFLASH mode with debugger.

	Step to run:
		- Step 1: Build example.
		- Step 2: Burn hex file into board (if run on ROM mode)
		- Step 3: Connect UART1->USB port on this board to USB port on your computer
		- Step 4: Configure hardware and serial display as above instruction
		- Step 5: Run example and observe on serial display

DeepSleep example

Example description:
	Purpose:
		This example describes how to enter system in DeepSleep mode and wake up it
		by using WIC Interrupt
	Process:
		Configure WIC to receive interrupt from WAKEUP0 pin
		Enter system in DeepSleep by using CLKPWR_DeepSleep function (call __WFI())
		Connect WAKEUP0 to 3.3V to generate interrupt and wake up the system.
