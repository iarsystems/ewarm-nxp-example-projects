@Example description:
	Purpose:
		This example describes how to enter system in PowerDown mode and wake up it
		by using EVRT Interrupt
	Process:
		Configure EVRT to receive interrupt from WAKEUP0 pin
		Enter system in PowerDown by using PWR_PowerDown function (call __WFI())
		Connect WAKEUP0 to 3.3V to generate interrupt and wake up the system.

@Directory contents:
	\IAR:	includes IAR EWARM project and configuration files
	lpc43xx_libcfg.h: Library configuration file - include needed driver library for this example
	Pwr_PowerDown.c: Main program

@How to run:
	Hardware configuration:
		This example was tested on:
			Hitex LPC4350 Board

	Running mode:
		RAM mode:   This example can be run on RAM mode with debugger.
		ROM mode:   This example can be run on EXFLASH mode with debugger.

	Step to run:
		- Step 1: Build example.
		- Step 2: Burn hex file into board (if run on ROM mode)
		- Step 3: Run example and observe the LEDs
		- Step 4: Connect WAKEUP0 pin to 3.3V to let the LEDs continue running
