@Example description:
	Purpose:
		This example describes how to write a USB DFU
		application using USB ROM driver on LPC18xx.


@Directory contents:
	\EWARM:	includes IAR project and configuration files 
	
	usbd_dfu.c: contains USB DFU function
	dfu_usbdesc.c : contains usb device descriptor.

@How to run:
	Hardware configuration:		
		This example was tested on:
			Hitex LPC1800 Development Board
	
	Running mode:
		RAM mode:   This example can be run on RAM mode with debugger. 
					All files must be build to .elf file, this file will be loaded into RAM through a 
					debugger tool before running 

		ROM mode:   This example can be run on EXFLASH mode with debugger. 
	
	Step to run:
		- Step 1: Build example.
		- Step 2: Burn hex file into board (if run on ROM mode)
		- Step 3: Configure hardware as above instruction 
		- Step 4: Run example
		- Step 5: Connect the USB0 port to PC with USB cable.
		- Step 6: - Testing DFU interface
			  		- On PC launch DFU download utility.
		
		(Pls see "LPC18xx Example Description" document - chapter "Examples > USBDEV_ROM > USBDFU"
		for more details)
		
@Tip:
	- Open <installation-root>\arm\examples\NXP\LPC18xx\LPC1800CMSIS\Examples\USBDEV_ROM\USBDEV_ROM.eww project file to run example on IAR
