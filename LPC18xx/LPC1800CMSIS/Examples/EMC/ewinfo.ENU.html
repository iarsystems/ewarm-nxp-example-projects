NOR Flash programing example

Example description:
	Purpose:
		This example describes how to use EMC to program the external Nor Flash
	Process:
		Setup Pin for External Memory
		Setup EMC timing on CS0
		Erase Sector 0 of Flash at address 0x1C000000
		Program the Menu1 content to Nor Flash
		Read Back and Print it to UART1

Directory contents:
	\EWARM:	includes IAR project and configuration files	
  	Emc_NorFlash.c: Main program
	FlashPrg.c: Flash Function

How to run:
	Hardware configuration:		
		This example was tested on:
			Hitex LPC1800 Board

	Running mode:
		RAM mode:	This example can run only on RAM mode.

	Step to run:
		- Step 1: Build example.
		- Step 2: Burn hex file into board (if run on ROM mode)
		- Step 3: Configure hardware and serial display as above instruction 
		- Step 4: Run example

Tip:
	- Open  \arm\examples\NXP\LPC18xx\LPC1800CMSIS\Examples\EMC\EMC.eww project file to run example on IAR

EMC SDRAM example

Example description:
	Purpose:
		This example describes how to use EMC to connect with external SDRAM
	Process:
		Indicating the board that example will run on by define in SDRAM_Init
		when run on NXP EA board,
			Set M3 Clock equal or lower than 84MHz by set mult<=7 in function CGU_Init()
		Setup Pin for External Memory
		Setup EMC timing on DYCS0
		Fill RAM content
		Check RAM content

Directory contents:
	\EWARM:	includes IAR project and configuration files	
  	Emc_Sdram.c: Main program
	SDRAM_Init.c: contains MT48LC4M32B2 and IS42S16400D SDRAM init function.

How to run:
	Hardware configuration:		
		This example was tested on:
			Hitex LPC1800 board

	Running mode:
		RAM mode:	This example can run RAM mode.
		ROM mode: 	This example can run EXFLASH mode.

	Step to run:
		- Step 1: Build example.
		- Step 2: Burn hex file into board (if run on ROM mode)
		- Step 3: Configure hardware and serial display as above instruction 
		- Step 4: Run example
		
Tip:
	- Open  \arm\examples\NXP\LPC18xx\LPC1800CMSIS\Examples\EMC\EMC.eww project file to run example on IAR
