Example description

  
This example describes how to config LCD on Hitex board and LogicPD 4.3.

Example description:
	Purpose:
		This example describes how to config LCD on Hitex board and LogicPD 4.3.
	Process:
		- Set up LCD panel clock
		- Init SDRAM and SRAM
		- Init IO port for LCD
		- Init LCD Controller
		- Init Touch Screen Controller
		- Init Hardware Cursor
		- Fill Colorbar and Logo
		*** Work with default jumper ***
					
Directory contents:
	\IAR:	includes IAR project and configuration files
	lpc18xx_libcfg.h: Library configuration file - include needed driver library for this example 
	IS42S16400D.c: Support functions for intializing I242S16400D IC
	lcd_qvga_tft.c and lcd_qvga_tft.h: Contains all functions to control LCD controller using SPI
	TSC2046.c and TSC2046.h:Contains all functions to control TSC2046 using SPI
	Cursor.c: Hardware Cursor Image in char array
	image.c: Logo Image in short array
	Lcd_Demo.c: Main program

How to run:
	Hardware configuration:		
		This example was tested on:
			Hitex LPC1850EVA-A4-2 Board
						
	Running mode:
		>IAR project:
			Built the project (F7)
			Internal SRAM mode:	This example can run on Internal SRAM mode.
				Download and debug (CTRL + D)
				Seclect Device Cortex M4 (if any)
			Hitex Flash mode: This example can run on Hitex Flash (external parallel flash SST39X320X), SPIFI quad SPI flash (with or without debug)
				Select "Project->Download->Download active application" to burn into flash

	Step to run:
		- Step 1: Build example.
		- Step 2: Burn hex file into board (if run on ROM mode)
		- Step 3: Run example and tap the LCD screen to move the cursor
