@Example description:
	Purpose:
		This example describes how to use EMC to program the external Nor Flash
	Process:
		Setup Pin for External Memory
		Setup EMC timing on CS0
		Erase Sector 0 of Flash at address 0x1C000000
		Program the Menu1 content to Nor Flash
		Read Back and Print it to UART0

@Directory contents:
	\Keil:	includes Keil project and configuration files
	\IAR:	includes IAR project and configuration files
  	Emc_NorFlash.c: Main program
	FlashPrg.c: Flash Function

@How to run:
	Hardware configuration:		
		This example was tested on:
			Hitex LPC1850EVA-A4-2 Board
			
		Serial display configuration:(e.g: TeraTerm, Hyperterminal, Flash Magic...) 
			– 115200bps 
			– 8 data bit 
			– No parity 
			– 1 stop bit 
			– No flow control
			– Connect UART0 on this board to COM port on your computer

	Running mode:
		>Keil project:
		Internal SRAM mode:	This example can run on Internal SRAM mode.
                SPIFI 64MB Debug mode:
			+ This example can be run on external SPIFI flash:
					- Burn by Keil's Load feature:
						a) Copy all things in ..\..\..\Tools\Flash\Keil_Binaries to
						   C:\Keil\ARM\Flash (Keil uVision 4)
						b) Open Keil->Configure for flash compiling->Compile->Select debugger->Press LOAD 
                SPIFI 128MB mode:
			+ This example can be run on external SPIFI flash:
					- Burn by Keil's Load feature:
						a) Copy all things in ..\..\..\Tools\Flash\Keil_Binaries to
						   C:\Keil\ARM\Flash (Keil uVision 4)
						b) Open Keil->Configure for flash compiling->Compile->Select debugger->Press LOAD

		>IAR project:
			Built the project (F7)
			Internal SRAM mode:	This example can run on Internal SRAM mode.
				Download and debug (CTRL + D)
				Seclect Device Cortex M4 (if any)
			Hitex Flash mode: This example can run on Hitex Flash (external parallel flash SST39X320X), SPIFI quad SPI flash (with or without debug)
				Select "Project->Download->Download active application" to burn into flash

	Step to run:
		- Step 1: Build example.
		- Step 2: Burn hex file into board (if run on Hitex Flash mode)
		- Step 3: Configure hardware and serial display as above instruction 
		- Step 4: Run example
