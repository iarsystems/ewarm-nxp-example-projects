
Purpose:
	The Audio project is a demo program for the LPC1343.
	It demonstrates an USB Audio Device - MicroPhone input.

Hardware configuration:
	This example verified on IAR-LPC1343 Rev.B board
	
Running mode:
	This example can run on ROM mode with a debugger or standalone after burning.

Note:		
	The USB Audio Device is recognized by the host PC running
	Windows which will load a generic Audio driver and add a
	microphone input which can be used for sound capture on the PC.
