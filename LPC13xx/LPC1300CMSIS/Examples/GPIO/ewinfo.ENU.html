GPIO_Interrupt

  Purpose:
	  This is a simple example for GPIO interrupt
	  Using UART0 with this configuration - 9600bps - 8 data bit - No parity - 1 stop bit - No flow control 
	  to display this conversion.

  Hardware configuration:
	  This example tested on IAR-LPC1343 Rev.B board
		  These jumpers must be configured as following:
		  J_Link_D: 		OFF
		  3.3V_IO_E: 		ON
		  3.3V_CORE_E: 	ON
		  PIO1_LOW:		OFF
		  PIO3_E:			ON
		  PIO3_HIGH:		OFF
		  PIO1_RS:		OFF
		  RST_E:			OFF
	  P0.1, P1.1, P2.1, P3.1 used for GPIO interrupt pin	

  Running mode:
	  This example can run on ROM mode with a debugger or standalone after burning.
	
  Note:
	  None

GPIO_LedBlinky

  Purpose:
	  A simple program to test GPIO functionality to drive LED

  Hardware configuration:
	  This example tested on IAR-LPC1343 Rev.B board
	  Red Led: 	P1.9
	  Green Led: 	P3.0
		  These jumpers must be configured as following:
		  J_Link_D: 		OFF
		  3.3V_IO_E: 		ON
		  3.3V_CORE_E: 	ON
		  PIO1_LOW:		OFF
		  PIO3_E:			ON
		  PIO3_HIGH:		OFF
		  PIO1_RS:		OFF
		  RST_E:			OFF

  Running mode:
	  This example can run on ROM mode with a debugger or stand-alone after burning.
	
  Note:
	  None
