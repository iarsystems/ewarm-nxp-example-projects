Example description

  
GPIO and Analog-To-Digital Converter (ADC)

DESCRIPTION
===========
  This lab demonstrates how to use the ADC module to read the value of the potentiometer.
  The three-color LED's color is set according to the value of the potentiometer and cycles
  through all the colors in the rainbow (red, orange, yellow, green, blue, and violet).

  This lab also demonstrates the GPIO interrupt capability on the KBD1 channel; if you press
  Switch 1, the LED turns on the red, green, and blue colors to produce a white light while
  the switch remains depressed.  Depressing switch 2 turns off the LED while the switch is
  engaged.  Pressing both switches simultaneously makes the LED cycle rapidly through all the
  colors.  The timing of the cycling is controlled by the delay( ) function.


GETTING STARTED
===============
  1) Start the IAR Embedded Workbench for ARM.

  2) Connect the on board OpenSDA debugger

  3) Press Ctrl+D or use Download and Debug button to start a debug session
