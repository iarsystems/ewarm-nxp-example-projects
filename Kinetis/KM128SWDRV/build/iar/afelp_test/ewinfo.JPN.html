Example description

  
DESCRIPTION
===========
   The Software Examples demonstrate correct use of the MKMxxZxx(A)Cxx5 bare metal
  software drivers in an application.  
  This example shows typical use of the Analogue Front End (AFE) peripheral operating
  in low power mode with PGA disabled. The device runs in BLPI mode with 1.0 MHz
  system, bus, flash and AFE clocks derived from the 4.0 MHz Internal Clock (IRC).
  
CONFIGURATION
=============
  
  Release: The example is flashed into the embedded flash memory of the KM34Z128A
  microcontroller.
  
  By default the project is configured to use an I-Jet debugger.
  
COMPATIBILITY
=============
  This example can be used with a TWR-KM34Z75M board. 
  The FreeMASTER communicates via UART2 that is configured for 19200/8-N-1 and
  serviced by application software in a polling mode. Connect USB port of your PC to
  the on-board OpenSDA's USB to serial bridge (J27) when running example on
  TWR-KM34Z75M board.
  
 For more details please read the 
 MKMxxZxxACxx5 Bare Metal Software Drivers Reference Manual.
