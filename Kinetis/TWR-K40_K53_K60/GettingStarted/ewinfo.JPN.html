Getting Started
    
DESCRIPTION
===========
  This example project shows how to use the IAR Embedded Workbench for ARM
 to develop code for TWR-K40/TWR-K53/TWR-K60/TWR-K70/TWR-K20D50/TWR-KL25Z48M
 boards. It shows basic use of GPIO and system timer.
  It toggles LED D16 (PC7) of TWR-K40 and TWR-K53, D16 (PA11) of TWR-K60,
 D7 (PTA4) of TWR-K70, D501(PTC9) of TWR-K20D50 and LED1 (D501)on TWR-KL25Z48M
 15 times per second.

  Note: The flashloader keep controller in unlocked state. For locking the controller
  only 00 (10 is unlock combination at address 0x40C bits 0,1) is valid combination


COMPATIBILITY
=============
  The example project is compatible with Freescale TWR-K40, TWR-K60, TWR-K53
  TWR-K70, TWR-K20D50 and KL25Z48M boards.
  By default, the project is configured to use the J-Link JTAG interface, expect
  TWR-K20D50 which uses  the on board PE debugger.

CONFIGURATION
=============
  The Workspace contains Getting Started example with few configurations.

  K40 RAM Debug:   The example is loaded into the internal RAM of K40
                   microcontroller.
  K40 Flash Debug: The example is flashed into the embedded flash memory of K40
                   microcontroller
  K60 RAM Debug:   The example is loaded into the internal RAM of K60
                   microcontroller.
  K60 Flash Debug: The example is flashed into the embedded flash memory of K60
                   microcontroller
  K53 RAM Debug:   The example is loaded into the internal RAM of K53
                   microcontroller.
  K53 Flash Debug: The example is flashed nto the embedded flash memory of K53
                   microcontroller
  K70 RAM Debug:   The example is loaded into the internal RAM of K70
                   microcontroller.
  K70 Flash Debug: The example is flashed into the embedded flash memory of K70
                   microcontroller
  K20 RAM Debug:   The example is loaded into the internal RAM of K20
                   microcontroller.
  K20 Flash Debug: The example is flashed into the embedded flash memory of K20
                   microcontroller
  KL25 RAM Debug:  The example is loaded into the internal RAM of KL25
                   microcontroller
  KL25 Flash Debug:The example is flashed into the embedded flash memory of KL25
                   microcontroller

 Jumpers:
  TWR-K53 board
  J17 - Filled
  J11 - 1-2
  J34 - 1-2

  TWR-K60 board
  J523 - 1-2

  TWR-K70 board
  J19 - Filled
  J18 - 1-2

  TWR-K20D50 board
  J11 - Filled
  J30 - 1-2
  
  TWR-KL25Z48M
  J18 (ELE_LED1) - filled

GETTING STARTED
===============

  1) Start the IAR Embedded Workbench for ARM.

  2) Connect the J-Link

  3) Power on the board.

  4) Select GettingStarted RAM or Flash Configuration

  5) Press Ctrl+D or use Download and Debug button to start a debug session
