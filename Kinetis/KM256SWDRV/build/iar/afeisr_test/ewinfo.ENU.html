Example description

  
DESCRIPTION
===========
   The Software Examples demonstrate correct use of the MKM34Z256VLx7 bare metal
  software drivers in an application. 
  This example shows typical use of the Analogue Front End (AFE) peripheral in
  software triggered and interrupt conversion mode. The FreeMASTER communication
  enables monitoring of the measured results as well as performing on-the-fly phase
  delay updates. 
 
CONFIGURATION
=============
  
  Release: The example is flashed into the embedded flash memory of the KM34Z256
  microcontroller.
  
  By default the project is configured to use an I-Jet debugger.
  
COMPATIBILITY
=============
  This example can be used with a TWR-KM34Z75M board. 
  The FreeMASTER interfaces via UART2 of the MKM34Z256VLx7 device which is
  configured for 38400/8-N-1 and handled by interrupts. When using
  TWR-KM34Z75M board, connect USB port of your PC to the on-board OpenSDA's USB
  to serial bridge (J27).

 For more details please read the 
 MKM34Z256VLx7 Bare Metal Software Drivers Reference Manual.
