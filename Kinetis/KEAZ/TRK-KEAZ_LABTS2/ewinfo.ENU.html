Example description

  
Flex Timer (FTM)

DESCRIPTION
===========
  This lab demonstrates how to use the FTM module to generate
 periodical interrupts of different frequencies. Each of the four leds
 is configured to toggle at a different frequency which is configurable
 using the FTM_SetPeriod() function.


CONFIGURATION
=============
  The Workspace contains example with few configurations:

  Flash Debug KEAZ128xxx4 : The example is flashed into the embedded flash.
  Flash Debug KEAZ64xxx4  : The example is flashed into the embedded flash.
  Flash Debug KEAZN16xxx2 : The example is flashed into the embedded flash.
  Flash Debug KEAZN32xxx2 : The example is flashed into the embedded flash.
  Flash Debug KEAZN64xxx2 : The example is flashed into the embedded flash.


GETTING STARTED
===============
  1) Start the IAR Embedded Workbench for ARM.

  2) Connect the on board OpenSDA debugger

  3) Press Ctrl+D or use Download and Debug button to start a debug session
