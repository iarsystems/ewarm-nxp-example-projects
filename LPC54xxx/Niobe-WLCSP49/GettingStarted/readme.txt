GettingStarted

DESCRIPTION
===========
   This example project shows how to use the IAR Embedded Workbench for ARM
  to develop code for Niobe-WLCSP49 Xpresso evaluation board. It shows basic
  use of I/O, RI timer and interrupt controller.
   It toggles Green  LED every 0.25s.

COMPATIBILITY
=============

  The example project is compatible with Niobe-WLCSP49 Xpresso evaluation board.

CONFIGURATION
=============

   The GettingStarted application is downloaded to internal flash or RAM memory
  depending of the project's configuration and executed.

  IRAM - runs project from internal RAM.

  IFLASH - loads project to internal flash.

GETTING STARTED
===============

  1) Build and download the program.

  2) Run the program.
