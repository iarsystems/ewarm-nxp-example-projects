Example description

  
DESCRIPTION
===========
  This example project shows how to use the IAR Embedded Workbench for ARM to
 develop code for the Pasat DIP50CAN LPC2194 and DEVB4050 development board.
 UART1 is used for communication with Terminal Emulator running on PC.
 SPI is used to program SPI 25C256 memory. 
 P0.10 is used as a Chip Select for SPI 25C256 memory
 I2C is used to program I2C 24C256 memory
   Note: P0.07 must be pulled up to 3.3V because Philips LPC2194 SPI peripheral
       requires it.
   Address range for serial memory is limited to 0-7FFH internally due to 
       limited RAM on LPC2194. 

 
A. Start Terminal Emulation program, set it to 9600Bd, 1Stop bit, no parity
B. Start the program from 0x200
C. The following text should be seen on the Terminal:
   TinyARM  SPI&I2C Programmer
   >Select command (Space,Backspace):
    Edit RAM
D. Use SPACE or BACKSPACE to see other options:
    Edit RAM          
    Copy 24C256 -> RAM
    Copy RAM -> 24C256
    Copy 25C256 -> RAM
    Copy RAM -> 25C256
E. If Edit RAM is selected then first specify start address and press ENTER.
   Then it is possible to edit byte. Press ENTER to accept the value and to
   advance to next address.
   Press ESC to abort editing.
F. Copy 24C256 -> RAM will copy the whole content of 24C256 memory to LPC2194
   internal RAM buffer.
G. Copy RAM -> 24C256 will copy internal RAM buffer of LPC2194 to 24C256 memory.
H. Copy 25C256 -> RAM will copy the whole content of 25C256 memory to LPC2194
   internal RAM buffer.
I. Copy RAM -> 25C256 will copy internal RAM buffer of LPC2194 to 25C256 memory.

COMPATIBILITY
=============
  The project is compatible with Pasat DIP50CAN LPC2194 and DEVB4050 development board.
  The project is by default configured to use the J-Link JTAG interface 
  
GETTING STARTED
===============

  1) Start the IAR Embedded Workbench for ARM.

  2) Select File->Open->Workspace...
     Open the following workspace:

     \arm\examples\NXP
     \LPC219x\TinyARMDIP50CAN\DIP50CAN-PGM\PGM.eww
     
  3) Select project and configuration.

  4) Power the board and connect the J-Link.
 
  5) Press CTRL+D or use Download and Debug button to start a debug session.
