Example description

  
The 'WDT' projectS are simple programs for the LPC1766 
using IAR LPC-1768-SK Evaluation Board and demonstrating 
the use of WDT in both reset and interrupt mode.

Example functionality:                                                   
 - Clock Settings:
   - XTAL    =            	12.00 MHz
   - SYSCLK  =           	  72.00 MHz
   - PCLK    = SYSCLK/4 = 	18.00 MHz

 - Program is used to read a character and to print out a string.
   115200 Baud, 1 Stop Bit, 8 Data Bits, No Flow Control
 
The WDT program can run from Internal RAM or Flash located on chip
(target debugging)

Note: 
 - remove ISP_E and RST_E jumper for UART0 to work properly
