Getting Started

DESCRIPTION
===========
   This example project shows how to use the IAR Embedded Workbench for ARM
  to develop code for MCIMX6Q-SDB board. IT shows basic use of I/O, Private Timer
  and GIC

COMPATIBILITY
=============
  The example project is compatible with the MCIMX6Q-SDB PDK board.
  By default, the project is configured to use the J-Link JTAG interface.

CONFIGURATIONS
===============
OCRAM_DEBUG
  Intended for debug in the on-chip RAM.

DDR_DEBUG
  Intended for debug in the DDR3 memory.

GETTING STARTED
===============
  1) Power the board and connect the debugger probe.

  2) Select a configuration and build it. Press CTRL+D or use Download and
     Debug button to start a debug session.

  3) Press F5 or use the GO button to run the example. The red user led on the
     board will blink.
